# Elevator Action Clone

A modern, browser-based 8-bit style clone of the classic 1980s arcade game "Elevator Action".

## Features
- Multi-floor navigation using elevators and escalators.
- Combat mechanics with enemy agents.
- Document collection system.
- Level progression with increasing difficulty.
- Retro aesthetics and sound effects.

## How to Play
1. Clone the repository.
2. Open `index.html` in your web browser.
3. Use the arrow keys to move and the spacebar to shoot (or as configured in-game).

## Technologies Used
- HTML5 Canvas
- Vanilla JavaScript
- CSS3
